 ## Um glossário com os principais conceitos aprendidos:
 
 Com base nos materiais do bootcamp e nas fontes complementares, aqui está um **glossário estruturado** com os principais conceitos essenciais para sua jornada em análise e engenharia de dados:

### 1. Fundamentos de Programação e Python
*   **Python:** Uma linguagem de programação popular e versátil, conhecida por sua **sintaxe clara e estruturada**, amplamente utilizada em servidores, análise de dados e automação.
*   **Sintaxe:** O conjunto de regras que define como os programas Python são escritos e interpretados.
*   **Variáveis e Tipos de Dados:** Recipientes para armazenar valores de dados, que podem ser números, strings (texto), booleanos (verdadeiro/falso) ou estruturas complexas.
*   **Estruturas de Dados Compostos:** Incluem **Lists** (listas indexadas e mutáveis), **Tuples** (imutáveis), **Sets** (conjuntos não ordenados) e **Dictionaries** (pares chave-valor).
*   **Programação Orientada a Objetos (POO):** Paradigma que utiliza **Classes e Objetos** para estruturar o código, permitindo conceitos como herança, polimorfismo e encapsulamento.
*   **Funções:** Blocos de código reutilizáveis que podem aceitar argumentos obrigatórios, opcionais ou arbitrários para executar tarefas específicas.

### 2. Ciência e Análise de Dados
*   **Análise de Dados:** O processo de **transformar dados brutos em insights** para apoiar decisões de negócios estratégicas e baseadas em evidências.
*   **Data Cleanup (Limpeza de Dados):** Técnica vital para remover erros, inconsistências, duplicatas e **outliers** (valores atípicos), garantindo que os conjuntos de dados sejam precisos e confiáveis.
*   **Data Collection (Coleta de Dados):** O ato de reunir informações de diversas fontes, como bancos de dados, planilhas, APIs ou *web scraping*.
*   **Data-Driven Decisions:** Decisões empresariais tomadas com base na interpretação de dados reais, em vez de apenas intuição.
*   **Engenharia de Dados:** Área focada na preparação de dados, automação de rotinas de análise e monitoramento de processos em larga escala.

### 3. Banco de Dados e Infraestrutura
*   **SQL (Structured Query Language):** Linguagem padrão para consultar, gerenciar e manipular bancos de dados relacionais.
*   **BigQuery:** Armazém de dados (data warehouse) do Google Cloud que permite ingerir, transformar e analisar grandes volumes de dados de forma rápida.
*   **AWS (Amazon Web Services):** Provedor de serviços em nuvem utilizado no bootcamp para o **monitoramento de processos** e infraestrutura de dados.
*   **IAM (Identity and Access Management):** Sistema para gerenciar usuários, funções e permissões de acesso a recursos na nuvem.

### 4. Inteligência Artificial e Machine Learning
*   **IA Generativa:** Tecnologia utilizada para criar soluções modernas, como **assistentes virtuais** capazes de responder dúvidas complexas sobre dados.
*   **Machine Learning (Aprendizado de Máquina):** Ramo da IA que utiliza algoritmos para reconhecer padrões em dados, realizar **modelagem preditiva** e prever tendências futuras.
*   **Gemini:** Conjunto de recursos baseados em IA integrados a ferramentas como o BigQuery para auxiliar no fluxo de trabalho de dados, desde a codificação até a visualização.
*   **NotebookLM:** Ferramenta avançada para organizar, resumir e consultar materiais técnicos de forma inteligente.

### 5. Visualização e Business Intelligence (BI)
*   **Data Visualization (Visualização de Dados):** Uso de gráficos e elementos visuais para ajudar stakeholders a interpretarem descobertas e tendências.
*   **Dashboard:** Painel visual interativo que consolida métricas e dados importantes para facilitar o monitoramento e a tomada de decisão.
*   **Looker / Looker Studio:** Ferramentas do ecossistema Google para explorar, analisar e criar relatórios e dashboards dinâmicos.
*   **LookML:** Linguagem de modelagem usada no Looker para definir dimensões, medidas e relações de dados de forma padronizada.
